Features

Critical CSS

Replace your css with critical css, then followed by deferred full css.

Lazyload Images

Automatically replace all of your images into lazyloaded images.

PWA Ready

Built in service worker configuration and manifest.json file.

Google Analytics

Google Analytics integration with latest gtag.js implementation.

Netlify Deployment

Optimized for Netlify Deployment. Netlify.toml included in the project.

Secure Headers

Best practices of using security headers. Integrated with netlify _headers file.

TailwindCSS

Optimized for TailwindCSS project. Compressed and minified using PurgeCSS.

Poi Bundler

Lightweight and Good DX bundler. Poi-preset-offline included in the project.
